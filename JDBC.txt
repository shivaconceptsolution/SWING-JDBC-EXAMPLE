JDBC?

JDBC means Java database connectivity, it will work as a bridge between appliation and database.

It use predefie driver to manage conection from application to multiple database server such as MYSQL, Oracle, MS-SQL, etc.


JDBC uses four different type of driver to establish connectio.

Type1:-   JDBC- ODBC


Type2:-    JDBC Vendor Specific


Type 3:-    JDBC Network Specific


Type 4:-     JDBC Client Server or Java thin Driver.


Type 4 is the most updated driver that is used to manage database connection from any database server to all platform.
